# ¿Qué es Visual Studio Code y para qué se utiliza?
---------------------------------------------------
1. Es un potente editor de código fuente desarrollado por Microsoft.
2. Se centra en la simplicidad y la eficiencia.

**Funcionalidades**
1. Facilita la edición y desarrollo de código fuente gracias a su interfaz intuitiva y sus herramientas inteligentes.
2. Identifica y corrige errores en las aplicaciones y scripts gracias a sus herramientas de depuración.
3. Gestiona proyectos y carpetas de trabajo de forma estructurada.
4. Facilita el seguimiento y control de versiones del código.
5. Utiliza las extenciones disponibles en el Marketplace.
6. Permite trabajar en equipo con otros desarrolladores gracias a Live Share.
7. Soporta una amplia variedad de lenguajes de programación y frameworks.
8. Ideal para desarrollar aplicaciones web.
