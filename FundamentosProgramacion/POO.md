## Pilares POO



### Lo que entendí: 
1. Abstracción: Aquí se limitan las características y funciones que se puedan hacer.
2. Encapsulación: Se limitan los datos que se pueden disponer de un objeto para que no hayan errores y así conntinuar con un hilo conductor y así controlar
cuándo y cuáles datos se puedan utilizar para obtener el funcionamiento esperado.
3. Herencia: Se copian los atributos y métodos incluso se pueden agregar más y así poder complementar o crear una nueva clase.
4. Polimorfismo: Un método podría tener muchas formas de usarse pero se podrían redefinir ciertos métodos nuevos, por ejemplo cambiar funciones y crear nuevas
clases o variables.
